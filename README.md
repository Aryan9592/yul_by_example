## Yul By Example 

Basically, we made [Solidity by example](https://solidity-by-example.org/) but for Yul :)<br><br>
**Required Solidity Version**: v0.8.17.<br>
**Required SPDX-License-Identifier**: MIT


To see the book change live run:

```sh
mdbook serve
```
